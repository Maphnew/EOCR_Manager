# EOCR_Manager

## Install nodejs & npm
- Search "install nodejs"
- Search "install npm"

### Download 

- https://github.com/Maphnew/EOCR_Manager.git  

### Install packages & Run server
```bash
$ npm install
$ npm run dev
```

- localhost:8000 접속


## Gateway 설치

```bash
$ git clone https://github.com/Maphnew/EOCR_Manager.git  
$ cd EOCR_Manager
$ npm install
$ npm run start
```
